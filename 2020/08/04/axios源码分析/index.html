axios 是什么

axios 是基于 Promise 并用于浏览器和 node.js 的 HTTP 请求库。

axios 的常见功能

  • 支持请求、响应拦截
  • 取消请求
  1. 通过 CancelToken.source 这个工厂函数来获取 cancel 函数。
  1. 通过 CancelToken 的构造函数来获取 cancel 函数。
  • 预防 XSRF

可以通过设置 XSRF token 来预防 XSRF/CSRF。

  • 支持 Promise API
  • 获取上传、下载进度

其实它的作用不外乎存放 cancel message，那为什么要再实现一个 toString 函数呢？
别忘了，当一个对象涉及到隐式转换的时候，一般会经历以下几个步骤：

  1. 首先调用 Symbol.toPrimitive，如果不存在就会走以下两个分支。
  2. 如果涉及到 string 类型的转换，会先后调用 toString 和 valueOf
  3. 如果涉及到 number 类型的转换，会先后调用 valueOf 和 toString
